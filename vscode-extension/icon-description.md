# Professional Context Engine Icon

## Design Description
The extension icon should be a 128x128 PNG image with the following design:

### Visual Elements
- **Background**: Circular gradient from blue (#4A90E2) to darker blue (#357ABD)
- **Main Symbol**: Database/context layers represented by three stacked ellipses in white with decreasing opacity
- **Connection Lines**: Vertical lines connecting the database layers
- **AI Indicator**: Small golden circle in the top-right with "AI" text

### Symbolism
- **Database Layers**: Represent structured context storage and organization
- **Connection Lines**: Show relationships and data flow
- **AI Badge**: Indicates intelligent, AI-powered functionality
- **Blue Gradient**: Professional, trustworthy, and technology-focused

### Technical Specifications
- **Size**: 128x128 pixels
- **Format**: PNG with transparency
- **Colors**: 
  - Primary: #4A90E2 (blue)
  - Secondary: #357ABD (darker blue)
  - Accent: #FFD700 (gold)
  - Text: #FFFFFF (white)

This icon should convey professionalism, intelligence, and data management capabilities while being easily recognizable in the VS Code marketplace and activity bar.
